Working with links

In this chapter we are going to practice the 4 types of links:

Linking to Other Sites (Absolute URL):
When you link to a different website, the value of the href attribute will be the full web address for the site, which is know as an absolute URL.
Linking to Other pages on the Same Site (Relative URL):
When you link to other pages within the same site, you do not need to specify the domain name in the URL. You can use a shorthand known as relative URL.

Linking to Fragment Identifiers (to a Specific Part of the Same Page or Another Page):
To link to an element that uses an id attribute, so the value of the href attribute starts with the "hash mark" # symbol followed by the value of the id attribute of the element you want to link to.

Uniform Resource Locator (URL)

A Uniform Resource Locator (URL), colloquially termed a web address, is a reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.

We have two types of links, "absolute link" and "relative link". Absolute link is for linking to another website while relative link is for linking to another page on the same website or to other part (section) within the same page. In relative links, we don't use the full URL address, we just need to specify the file name with its full local path if needed.
