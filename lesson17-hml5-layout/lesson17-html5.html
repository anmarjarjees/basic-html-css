HTML5 New Elements

The semantic meaning

HTML5 introduces a new set of elements that allow you to divide up the parts of the page.

ID Attribute

Any HTML element can have an id attribute, this id attribute is used to uniquely identify an element on the page. The id symbol is #. "id" attribute can be used for linking to a specific location within the same page or a location within another page, it can also be used in styling rules, or by JavaScript code.

id is used to uniquely identify an element from other elements on the page. You can write any name for the id but without any spaces or special characters only "dash " or "underscore ".

Please be advised that the name (the value) of the id can be used only for one time in the same page.

The attribute "id " can be used for linking to a specific location within the same page, or for styling using "CSS ".

Class Attribute

Every HTML element can also have a class attribute, so it's also a global attribute like id. Note like "id", class value can be used more than one time in the same page.

In case if you need to use the same name or the same value "important-paragraph " for more than one time (for more than one paragraph), in such case we can use the other attribute named "class ". class value can be used as many time as we need.
